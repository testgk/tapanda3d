# ta3d

A Panda3D-based terrain and entity management game engine.

## Description

This project uses the Panda3D game engine to create a 3D terrain visualization and entity management system. It includes:

- Terrain rendering and physics
- Camera controls
- Entity management and selection
- Collision detection
- Physics simulation using Bullet

## Repository

This repository is maintained on GitHub and mirrored from GitLab.

## Requirements

- Python 3.x
- Panda3D
- Bullet physics engine

## Running

```bash
python main.py
```

## Features

- Dynamic terrain loading with multiple heightmaps
- Physics-based entity simulation
- Camera controls for terrain navigation
- Entity selection and management
- Collision detection and physics layers
